## Requisitos

- [Python 3.13](https://www.python.org/downloads/)
- [Git](https://git-scm.com/downloads)

## Instalação

```bash
# 1. Clone o repositório
git clone https://github.com/ShiniAkaiUsagi/rocketseat_python_flask_api.git

# 2. Acesse a pasta do projeto
cd rocketseat_python_python_flask_api

# 3. Execute o script de build para preparar as ferramentas e ambiente
sh scripts/build.sh

# O script executa:
# python.exe -m pip install --upgrade pip
# pip install -U poetry
# poetry install
# poetry run pre-commit run
```

## Executando a aplicação
```bash
# Para executar o código da Agenda, basta utilizar o comando abaixo:
poetry run python src/desafio_agenda.py
```
